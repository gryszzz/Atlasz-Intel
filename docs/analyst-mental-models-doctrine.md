# Analyst Mental Models Doctrine

The next capability jump is not more tools. It is better models of how the
world works, how information moves, how systems fail, and how incentives shape
behavior.

The corpus contract lives in `src/agentMasteryCorpus.ts`.

## Core Dimensions

Atlasz should reason across:

- systems
- networks
- incentives
- evidence
- relationships
- history
- security
- data pipelines

This is what turns isolated feeds into connected intelligence.

## OPSEC

OPSEC is defensive: protect systems and information, minimize data exposure,
model attribution risk, and keep privacy/security architecture explicit.

Key sources:

- https://github.com/pluja/awesome-privacy
- https://github.com/privacyguides/privacyguides.org
- https://github.com/OWASP/CheatSheetSeries

## Analyst Standard

No source means no claim. One source means one attributed claim. Multiple
independent sources can increase confidence, but still need uncertainty labels.
Graph reachability is not causality. Historical analogy is not proof. Economic
context is not prediction.
